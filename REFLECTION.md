# Reflection

## What did you learn about using Git and GitHub?
I learned that Git is imporant for team project where each individual done their works locally, but they need a universal platform to put their works together.

## What was most confusing or frustrating?
The most confusing part was learning which commands to use in PowerShell, like touch that are not available in PowerShell.

## Why do you think developers use Git?
Developers use Git so they can save progress, track changes, go back to older versions, and work with others more easily.

## How might this help with larger projects in the future?
For larger projects, Git can help organize work into smaller commits, to review, and point out each others' mistakes in peer review.
